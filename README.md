## Getting Started

1. Clone the repository:

```bash
git clone [your-repo-url]
cd ai-software-engineering-assignment
```

2. Install Python dependencies for Task 2 and Task 3:

```bash
pip install -r part2_practical/task2_automated_testing/requirements.txt
pip install pandas scikit-learn joblib matplotlib
```

3. Run notebooks or scripts as needed:

* Task 2: `pytest` tests
* Task 3: Open `notebook.ipynb` in Jupyter or Colab
